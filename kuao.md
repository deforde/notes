# ld

-------------------------
Examine runtime linker behaviour using the following:
```
LD_DEBUG=files LD_DEBUG_OUTPUT=ld_debug_out app
```
There are other useful options for `LD_DEBUG` as well, like `bindings` etc. (see `man ld.so` for more info).
