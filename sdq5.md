# Basic example:

Content:
```
\documentclass{article}
\begin{document}
This is a test.
\end{document}
```
To compile to pdf:
`pdflatex test.tex`
Or:
`xelatex test.tex`

-------------------------------------------

# Equations:

```
\documentclass{article}
\begin{document}
\begin{equation}
E = mc^2
\end{equation}
\end{document}
```
